# Frankie's Dotfiles

## Get started

Clone the repository

```shell
git clone https://github.com/frankjlin16/dotfiles.git
```

Run the symlink script

```shell
cd ~/dotfiles
chmod +x makesymlinks.sh
./makesymlinks.sh
```
